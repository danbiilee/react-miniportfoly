**미니포트폴리**는 리액트로 개발한 포트폴리오 및 이력서 용도의 웹 사이트입니다.

- **Concept**: 싸이월드 미니홈피의 레이아웃, 디자인 및 감성을 담았습니다. 
- **Contents**: 이력과 자기 소개, 포트폴리오에 대한 내용을 담았습니다.
